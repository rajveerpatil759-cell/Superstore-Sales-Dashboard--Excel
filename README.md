# Superstore USA Sales Dashboard (Excel)

##  Project Overview
This project presents an **interactive Excel dashboard** built using the Superstore USA sales dataset.  
The dashboard provides insights into **sales performance, profitability, orders, discounts, and regional trends** to support data-driven decision making.

---

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Formatting
- Dashboard Design

---

##  Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Profit Margin

---

## Dashboard Features
- Sales by Category & Sub-Category
- Profit by Sub-Category
- Sales by Region and State (Map View)
- Monthly Sales & Profit Trend
- Discount vs Profit Margin Analysis
- Interactive filters for:
  - Year
  - Segment
  - Category

##  Conclusion
This Excel dashboard successfully transforms raw Superstore sales data into meaningful insights.  
By analyzing sales, profit, discounts, and regional performance, the dashboard highlights key business trends and problem areas such as low-margin and loss-making sub-categories.

The project demonstrates practical skills in **data analysis, visualization, and dashboard design using Excel**, making it suitable for entry-level roles such as **Data Analyst, MIS Executive, or Business Analyst**.  
Overall, this project showcases the ability to convert data into actionable insights and communicate them effectively through a well-structured dashboard.
